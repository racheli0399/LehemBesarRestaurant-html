שום קונפי ושמן ירוק. בליווי פלוט לחם

לחמעג'ון כבש // 65

סלט פרגיות // 74

עגבניות שרי, מלפפון, בצל ירוק, קוביות בטטה וסלק. בתיבול וויניגרט הדרים


רוסטביף // 58

טרטר דג לבן // 58
